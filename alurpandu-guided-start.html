Delivery

Pastikan project benar-benar siap dikirim

Gunakan checklist di bawah ini untuk meminimalkan miskomunikasi dan memastikan serah terima berkas berjalan lancar:

  • Preview link sudah ada (berisi update draf visual atau fungsional terbaru)
  • Feedback client sudah dicatat (seluruh masukan tertulis sebelumnya telah diakomodasi)
  • Revisi sudah selesai (bebas bug mayor, typo, atau kesalahan rendering draf)
  • Final file link tersedia (tautan download folder awan terkompresi/tidak rusak)
  • Source/raw file jelas jika termasuk (file mentahan terorganisir dengan penamaan layer rapi)
  • Handover notes siap (penjelasan singkat mengenai cara pakai, struktur aset, atau panduan hosting)
  • Client confirmation tercatat (persetujuan tertulis bahwa deliverables sudah sesuai ekspektasi)

Invoice & Payment

Jangan berhenti di invoice sent

Invoice sent belum berarti payment aman. Catat invoice status, due date, next follow-up, dan payment status secara berkala di ledger Anda.

1. Tanggal Jatuh Tempo (Due Date)

Berikan batas waktu pembayaran yang jelas (misalnya: Net 7, Net 14). Catat tanggal jatuh tempo ini di dashboard agar pengingat otomatis berjalan.

2. Rencana Follow-up Lanjutan

Jadwalkan pesan tindak lanjut ramah 1 hari sebelum jatuh tempo, hari-H, dan 3 hari setelah jatuh tempo jika belum ada konfirmasi pembayaran.

3. Status Rekonsiliasi Kas

Setelah dana terverifikasi masuk ke rekening Anda, segera ubah status di AlurKarya menjadi Paid dan serahkan file final secara resmi.

Help

Kalau bingung, mulai dari sini

Beberapa pertanyaan teknis yang sering ditanyakan seputar pengoperasian AlurKarya:

Project tidak muncul di dashboard
Periksa apakah project sudah didaftarkan di workspace board. Project baru harus memiliki minimal satu task aktif agar muncul di tracker mingguan Anda.
Checklist tidak tersimpan saat refresh halaman
AlurPandu menyimpan progress di database browser lokal Anda (localStorage). Pastikan mode private browsing/incognito Anda tidak otomatis menghapus cache saat tab ditutup, atau coba bersihkan ruang penyimpanan browser Anda.
Template peran (role) belum sesuai dengan kebiasaan kerja
Template peran di atas adalah panduan umum. Di dalam AlurKarya, Anda bebas menambahkan sub-checklist kustom, menghapus tag bawaan, dan menyusun urutan kolom Kanban sesuai alur spesifik workflow mandiri Anda.
Link tautan penting tidak dapat diklik atau error
Pastikan Anda menginput format URL secara lengkap saat mendaftarkan tautan di database card (misalnya gunakan format lengkap https://figma.com/... atau https://drive.google.com/... bukan hanya teks biasa).
Client Portal tidak menunjukkan pembaruan terbaru
Semua pembaruan data di dashboard kerja Anda disinkronkan secara instan ke tautan portal luar. Client cukup me-refresh halaman web portal mereka untuk melihat pembaruan status dan berkas terbaru.
Bagaimana cara menghapus seluruh data checklist ini?
Anda dapat menggunakan tombol "Reset Progress" yang berada di bagian kanan atas modul "Setup Workspace" untuk membersihkan progress lokal Anda di browser ini.
